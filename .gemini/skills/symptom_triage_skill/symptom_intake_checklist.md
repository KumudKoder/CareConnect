# Symptom Intake Checklist

Use short questions; ask only what changes triage decisions.

## Core questions
1. What symptom is bothering you most right now?
2. When did it start?
3. On a 0–10 scale, how severe is it now?
4. Is it getting better, worse, or staying the same?
5. Any associated symptoms? (fever, cough, breathlessness, chest pain, vomiting, rash, dizziness, confusion)

## Risk modifiers
6. Age group? (child, adult, older adult)
7. Pregnant or recently postpartum?
8. Existing conditions? (heart/lung disease, diabetes, hypertension, kidney disease, low immunity)
9. Current medicines and any known allergies?
10. Recent travel, sick contact, injury, new foods/medicines?

## Decision hints
- Severe + worsening + risk modifiers => urgent/same-day care.
- Mild + stable + no red flags => home care with monitoring.
- Any red flag => emergency now.

## Conversation style
- Empathetic and calm.
- One question at a time if user seems distressed.
- Keep to plain language; avoid jargon.
